# QuickStore

Simple local key-value store based on YAML::Store.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'quick_store'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install quick_store

## Usage

```ruby
require 'quick_store'
```
